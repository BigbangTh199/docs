# راه‌اندازی سریع برنامه‌های N8N
<hr className="mb-2" />

<a href="https://n8n.io/" className="blue-link">N8N</a> یک ابزار اتوماسیون و یکپارچگی گردش‌کار متن‌باز است که به کاربران این امکان را می‌دهد تا به راحتی اتوماسیون‌های پیچیده را ایجاد کنند. با استفاده از رابط بصری خود، n8n به شما اجازه می‌دهد تا با کشیدن و رها کردن، کارها و ارتباطات میان خدمات مختلف را تنظیم کنید. این پلتفرم از اتصال به تعداد زیادی از سرویس‌های وب، APIها و پایگاه‌داده‌ها پشتیبانی می‌کند و برای اتوماسیون وظایف و فرآیندهای کسب‌وکار بسیار مناسب است.







<div className="h-4" />
