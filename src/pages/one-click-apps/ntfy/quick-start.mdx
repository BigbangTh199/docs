# راه‌اندازی سریع برنامه‌های Ntfy.sh
<hr className="mb-2" />

<a href="https://ntfy.sh/" className="text-[#2196f3] ">Ntfy.sh</a> یک سرویس Open Source برای ارسال اعلان‌ها و نوتیفیکیشن‌ها به دستگاه‌های مختلف است. این ابزار به کاربران اجازه می‌دهد از طریق یک رابط ساده HTTP یا اپلیکیشن موبایل، پیام‌ها و هشدارها را ارسال کنند. Ntfy.sh از پروتکل‌های استاندارد مانند WebSockets و HTTP پشتیبانی می‌کند و با برنامه‌های اتوماسیون و اسکریپت‌ها به راحتی ادغام می‌شود. کاربران می‌توانند پیام‌های نوتیفیکیشن را به‌صورت فوری دریافت کرده و آن‌ها را بر اساس موضوعات مختلف دسته‌بندی کنند. این برنامه به‌صورت ایمن و با رمزنگاری قابل تنظیم است و امکان میزبانی بر روی سرورهای شخصی را نیز فراهم می‌کند. Ntfy.sh برای توسعه‌دهندگان و تیم‌هایی که به دنبال راه‌حلی ساده و سریع برای ارسال اعلان‌های خودکار هستند، ایده‌آل است.


<div className="h-4" />
