# راه‌اندازی سریع برنامه‌های Apache Answer
<hr className="mb-2" />

<a href="https://github.com/apache/incubator-answer" className="text-[#2196f3] ">Apache Answer</a> یک پلتفرم رایگان و متن‌باز Q&A (پرسش و پاسخ) است که می‌تواند برای هرتیمی در هر مقیاسی مناسب باشد. فرقی ندارد که Apache Answer یک انجمن باشد، یک help center یا یک پلتفرم اشتراک دانش (مانند StackOverFlow)؛ شما می‌توانید همواره روی Apache Answer حساب کنید.
<div className="h-4" />

برای راه‌اندازی برنامه‌ی آماده Apache Answer در لیارا، باید در صفحه <a href="https://console.liara.ir/apps/create" className="text-[#2196f3] ">ایجاد برنامه‌ کنسول لیارا</a> وارد بخش <b>برنامه‌های آماده</b> شوید.
سپس برنامه‌ی <b>Apache Answer</b> 
را انتخاب کنید و در صورت تمایل یک شناسه برای آن قرار دهید (در غیر این‌صورت، لیارا به صورت پیش‌فرض یک شناسه برای برنامه‌تان انتخاب می‌کند)؛ سپس بر روی گزینه **نصب سریع** یا **نصب پیشرفته** کلیک کنید تا برنامه‌تان ساخته شود:
