# راه‌اندازی سریع برنامه‌های PgHero
<hr className="mb-2" />

<a href="https://github.com/ankane/pghero" className="text-[#2196f3] ">PgHero</a> یک ابزار متن‌باز برای پایگاه‌داده‌های PostgreSQL است که به مدیران و توسعه‌دهندگان کمک می‌کند تا عملکرد پایگاه‌داده را بهینه‌سازی و مانیتور کنند. این ابزار به صورت دقیق شاخص‌هایی مانند کندی کوئری‌ها، مصرف منابع، استفاده از ایندکس‌ها، و قفل‌ها را نشان می‌دهد. PgHero به کاربران این امکان را می‌دهد تا مشکلات عملکردی پایگاه‌داده را به سرعت شناسایی کنند و کوئری‌های بهینه‌سازی نشده را پیدا کرده و بهبود دهند. با استفاده از این ابزار، تیم‌ها می‌توانند عملکرد PostgreSQL را بدون نیاز به استفاده از ابزارهای پیچیده، رصد کرده و بهبود بخشند.


<div className="h-4" />

برای راه‌اندازی برنامه‌ی آماده PgHero در لیارا، باید در صفحه <a href="https://console.liara.ir/apps/create" className="text-[#2196f3] ">ایجاد برنامه‌ کنسول لیارا</a> وارد بخش <b>برنامه‌های آماده</b> شوید.
سپس برنامه‌ی <b>PgHero</b>
