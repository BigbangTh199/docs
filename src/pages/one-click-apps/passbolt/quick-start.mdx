# راه‌اندازی سریع برنامه‌های Passbolt
<hr className="mb-2" />

<a href="https://www.passbolt.com/" className="text-[#2196f3] ">Passbolt</a> یک نرم‌افزار مدیریت رمز عبور Open Source است که برای تیم‌ها و سازمان‌ها طراحی شده تا به‌صورت امن رمزهای عبور و اطلاعات حساس را ذخیره و به اشتراک بگذارند. این برنامه از رمزنگاری کلید عمومی (PGP) استفاده می‌کند، به طوری که فقط کاربران مجاز قادر به دسترسی به داده‌ها هستند. Passbolt از ویژگی‌های متنوعی مانند مدیریت کاربران، تعیین سطوح دسترسی، و اشتراک‌گذاری امن اطلاعات برخوردار است. همچنین با ابزارهای متعددی مانند LDAP و Active Directory ادغام می‌شود و قابل نصب بر روی سرورهای شخصی یا استفاده به‌صورت ابری است. Passbolt به دلیل امنیت بالا و قابلیت‌های همکاری تیمی، انتخابی مناسب برای سازمان‌هایی است که نیاز به مدیریت امن رمزهای عبور دارند.<div className="h-4" />

برای راه‌اندازی برنامه‌ی آماده Passbolt در لیارا، باید در صفحه <a href="https://console.liara.ir/apps/create" className="text-[#2196f3] ">ایجاد برنامه‌ کنسول لیارا</a> وارد بخش <b>برنامه‌های آماده</b> شوید.
سپس برنامه‌ی <b>Passbolt</b> 
را انتخاب کنید و در صورت تمایل یک شناسه برای آن قرار دهید (در غیر این‌صورت، لیارا به صورت پیش‌فرض یک شناسه برای برنامه‌تان انتخاب می‌کند)؛ سپس بر روی گزینه **نصب سریع** یا **نصب پیشرفته** کلیک کنید تا برنامه‌تان ساخته شود.
