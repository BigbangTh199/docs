# راه‌اندازی سریع برنامه‌های Kibana
<hr className="mb-2" />

<a href="https://www.elastic.co/kibana" className="text-[#2196f3] ">Kibana</a> یک ابزار بصری‌سازی و تحلیل داده‌های متنی است که به‌ویژه با Elasticsearch ادغام می‌شود. این ابزار به کاربران امکان می‌دهد تا داده‌ها را به‌صورت نمودارها، گراف‌ها و نقشه‌ها نمایش دهند و به تحلیل و جستجوی پیشرفته بپردازند. Kibana به‌طور گسترده در مدیریت لاگ‌ها و مانیتورینگ سیستم‌ها استفاده می‌شود و رابط کاربری ساده و قدرتمندی را برای تحلیل داده‌ها فراهم می‌کند.

<div className="h-4" />

برای راه‌اندازی برنامه‌ی آماده Kibana در لیارا، باید در صفحه <a href="https://console.liara.ir/apps/create" className="text-[#2196f3] ">ایجاد برنامه‌ کنسول لیارا</a> وارد بخش <b>برنامه‌های آماده</b> شوید.
سپس برنامه‌ی <b>Kibana</b> 
را انتخاب کنید و پس از انتخاب شناسه یکتا برای برنامه، <a href="/paas/details/private-network" className="text-[#2196f3] ">شبکه خصوصی</a>، <a href="/paas/details/plans/about" className="text-[#2196f3] ">منابع سخت‌افزاری و بسته امکانات</a>، بر روی گزینه <b>ایجاد برنامه</b> کلیک کنید تا برنامه‌تان، ساخته شود:
