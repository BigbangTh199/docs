# راه‌اندازی سریع برنامه‌های Headless Chrome
<hr className="mb-2" />

<a href="https://developer.chrome.com/blog/headless-chrome" className="blue-link">Headless Chrome</a> نسخه‌ای از مرورگر گوگل کروم است که بدون رابط کاربری گرافیکی (GUI) اجرا می‌شود. این ویژگی امکان اجرای خودکار تست‌ها، پیمایش صفحات وب و انجام وظایف مختلف مرورگری را فراهم می‌کند، بدون اینکه نیاز به نمایش بصری صفحه باشد. از headless Chrome معمولاً در محیط‌های توسعه و تست خودکار استفاده می‌شود.
<div className="h-4" />

برای راه‌اندازی برنامه‌ی آماده Headless Chrome در لیارا، باید در صفحه <a href="https://console.liara.ir/apps/create" className="blue-link">ایجاد برنامه‌ کنسول لیارا</a> وارد بخش <b>برنامه‌های آماده</b> شوید.
سپس برنامه‌ی <b>Headless Chrome</b> 
را انتخاب کنید و پس از انتخاب شناسه یکتا برای برنامه، <a href="/paas/details/private-network" className="blue-link">شبکه خصوصی</a>، <a href="/paas/details/plans/about" className="blue-link">منابع سخت‌افزاری و بسته امکانات</a>، بر روی گزینه <b>ایجاد برنامه</b> کلیک کنید تا برنامه‌تان، ساخته شود:
