# استقرار برنامه‌های NuxtJS SSG در لیارا
<hr className="mb-2" />
<a href="https://nuxt.com/" className="blue-link">NuxtJS</a> یک فریم‌ورک مبتنی بر Vue است که بسیاری از قابلیت‌ها مانند SSR را برای شما به ارمغان می‌آورد. شما می‌توانید برنامه‌های NuxtJS خود را با <a href="../how-tos/create-app" className="blue-link">ایجاد برنامه‌های Static</a> بر روی لیارا دیپلوی کنید.

<div className="h-2" />
<Alert variant="success">
<p>
همچنین بخوانید: <a href="/paas/nodejs/related-apps/nuxtjs" className="blue-link">استقرار برنامه‌های NuxtJS با پلتفرم NodeJS لیارا</a>
</p>
</Alert>
<div className="h-2" />

برای گرفتن خروجی استاتیک در برنامه‌های NuxtJS بایستی مقدار <Important>target</Important> را در فایل <Important>nuxt.config.js</Important> برابر با <Important>static</Important> قرار دهید:

<div className="h-2" />
<div dir='ltr'>
    <Highlight className="js">
        {`export default {
  target: 'static'
}`}
    </Highlight>
</div>
<div className="h-2" />

درنهایت با هر بار اجرای دستور <Important>nuxt generate</Important>، خروجی استاتیک برنامه‌ی شما اعم از فایل‌های HTML , CSS , JavaScript در مسیر <Important>dist</Important> قرار خواهد گرفت. همچنین توجه داشته باشید که در این روش تمام درخواست‌های مورد نیاز برای دریافت اطلاعات از API ها در زمان اجرای دستور <Important>nuxt generate</Important> ارسال شده و اطلاعات دریافت شده نیز در پوشه‌ای با نام static در مسیر dist قرار داده می‌شود.
حال، برای استقرار برنامه خود در لیارا، کافیست تا وارد دایرکتوری <Important>dist</Important> شوید و با اجرای دستور زیر، برنامه خود را در لیارا، مستقر کنید:

<div className="h-2" />
<div dir='ltr'>
    <Highlight className="js">
        {`liara deploy --platform=static`}
    </Highlight>
</div>
