# اضافه کردن دامنه خریداری شده به برنامه

<hr className="mb-2" />

برای اتصال دامنه به برنامه مورد نظرتان در لیارا باید وارد برنامه شوید و از منوی سمت راست، روی گزینه‌ی **دامنه‌ها** کلیک کنید. سپس روی گزینه‌ی **افزودن دامنه** کلیک کرده و نام دامنه‌تان (مثلاً: mydomain.com) را وارد کنید. درنهایت با کلیک روی گزینه‌ی **ایجاد دامنه**، دامنه شما به برنامه اضافه می‌شود و همچنین به صفحه‌ی مدیریت دامنه هدایت می‌شوید.

<div className="h-4" />
<img src="https://files.liara.ir/liara/docs/add-domain.gif" alt="add-domain"/>
<div className="h-4" />

در این مرحله باید رکوردهای DNS درخواست شده را با استفاده از یک سرویس مدیریت DNS مانند <a href="https://www.cloudflare.com/" className="blue-link">Cloudflare</a> بر روی دامنه‌تان تنظیم کنید. 
درنهایت پس از ثبت رکوردهای DNS می‌توانید بر روی گزینه‌ی  **بررسی وضعیت رکوردها** کلیک کنید و با تغییر تیک‌های قرمز رنگ به سبز، یعنی اتصال دامنه‌ی شما به لیارا با موفقیت صورت گرفته است:

<div className="h-4" />
<img src="https://files.liara.ir/liara/docs/activated-domain.png" alt="activated-domain"/>
<div className="h-4" />

در حین ثبت دامنه، نکات زیر را حتماً در نظر بگیرید:
<ul>
    <li>درصورت استفاده از CDN باید گواهی SSL را نیز از سرویس‌دهنده‌ی DNS فعلی‌تان تهیه کنید.</li>
    <li>درصورت استفاده از CDN می‌توانید تیک قرمز رنگ و خطای تنظیم نشدن رکوردها را نادیده بگیرید.</li>
</ul>

</p>
</Alert>
