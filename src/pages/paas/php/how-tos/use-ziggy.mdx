# استفاده از Ziggy در برنامه‌های Laravel

Ziggy یک پکیج لاراول است که به شما امکان می‌دهد تا به راحتی به مسیر (route)های تعریف‌شده در لاراول از طریق جاوااسکریپت دسترسی داشته باشید. این ابزار برای توسعه‌دهندگانی که از فریم‌ورک‌های جاوااسکریپت سمت کاربر مانند <a href="../../vue/getting-started" className="blue-link">Vue</a> یا <a href="../../vue/getting-started" className="blue-link">React</a> استفاده می‌کنند، بسیار مفید است.
