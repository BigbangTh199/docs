# سرویس ذخیره‌سازی ابری لیارا
<hr className="mb-2" />

<Section id='what-is-object-storage' title='ذخیره‌سازی ابری چیست؟' />

ذخیره‌سازی ابری مبتنی بر object یا Object Storage نوعی سیستم ذخیره‌سازی داده‌هاست که در آن اطلاعات و داده‌ها به صورت object ذخیره می‌شوند. هر object شامل داده‌ها، متاداده و یک شناسه یکتاست که به آن امکان می‌دهد در مقیاس بزرگ، به‌طور موثر مدیریت شود. این روش ذخیره‌سازی برای مدیریت حجم بالای داده‌های غیرساختاریافته مانند فایل‌های چندرسانه‌ای و داده‌های بک‌آپ، ایده‌آل است و قابلیت انعطاف‌پذیری و مقیاس‌پذیری بالایی دارد. از مهم‌ترین ویژگی‌های آن می‌توان به دسترسی از راه دور و یکپارچگی آن، اشاره کرد.

<Section id='what-is-bucket' title='یک باکت (bucket) چیست؟' />
در سیستم‌های Object Storage، یک bucket (سطل)، مانند یک ظرف برای ذخیره و سازماندهی objectها عمل می‌کند. هر سطل شامل مجموعه‌ای از objectها است و به هر کدام از این objectها با یک کلید (شناسه یکتا) می‌توان دسترسی پیدا کرد.

<Section id="liara-object-storage" title="سرویس ذخیره‌سازی ابری لیارا" />
سرویس ذخیره‌سازی ابری لیارا، با سازگاری کامل با پروتکل S3، به شما این امکان را می‌دهد که حجم زیادی از داده‌ها را به سادگی، مدیریت کنید و آن‌ها را به کاربران‌تان ارائه دهید.
<div className="h-2" />

با استفاده از API ساده و سازگار با AWS SDK، می‌توانید به راحتی و در زبان‌ها و فریم‌ورک‌های مختلف برنامه‌نویسی، فایل‌های خود را مدیریت کنید. این سرویس با افزایش مقیاس‌پذیری برنامه‌ها، عملکرد فوق‌العاده‌ای را در مواجهه با workloadهای بالا ارائه می‌دهد و با یک کلیک، امکان تغییر سطح دسترسی فایل‌ها از عمومی به خصوصی را فراهم می‌سازد.
<div className="h-2" />

لیارا با رابط کاربری اختصاصی و سازگاری با AWS SDK، مدیریت ذخیره‌سازی ابری را برای شما آسان و امن می‌کند، تا با خیالی آسوده به توسعه برنامه‌های خود بپردازید.
